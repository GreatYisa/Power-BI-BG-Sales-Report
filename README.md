# Power-BI-BG-Sales-Report

### Project Description
- This project shows the Analytics of the BG Sales and covers the following.
  1. Data Cleaning
  2. PBI Dashboard
  3. Impact of the data on Annual sales by selling area, average sales by selling area etc.

### Technology used
  - Power BI
